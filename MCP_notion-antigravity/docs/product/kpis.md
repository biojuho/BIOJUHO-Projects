# KPI Baseline

Primary operating metrics for the Antigravity Content Engine:

- Brief generation success rate: target `>= 95%`
- Duplicate publication rate: target `< 1%`
- Dashboard refresh success rate: target `>= 99%`
- MCP canonical response validity: target `100%`
- Manual interventions per week: target `<= 1`
- Notion sync latency after brief generation: target `<= 15 minutes`

Suggested review cadence:

- Daily: run health, partial failures, approval backlog
- Weekly: report throughput, duplicate suppression, publishing latency
- Monthly: tool adoption, channel conversion, operator workload
