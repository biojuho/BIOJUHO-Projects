# DeSci Platform

탈중앙화 과학(Decentralized Science) 플랫폼

## Tech Stack
- **Backend:** Python (FastAPI)
- **Frontend:** React (Vite) + Tailwind CSS

## Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
