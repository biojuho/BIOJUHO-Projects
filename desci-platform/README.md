## Tech Stack
- **Backend:** Python (FastAPI) + Firebase Admin
- **Frontend:** React (Vite) + Tailwind CSS
- **Auth:** Firebase Authentication (Google + Email/PW)

## Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
