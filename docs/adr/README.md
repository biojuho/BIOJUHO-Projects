# Architecture Decision Records (ADR)

이 디렉터리에는 프로젝트의 주요 기술 결정을 기록합니다.

## 작성 규칙

1. 파일명: `NNNN-제목.md` (예: `0001-shared-llm-module.md`)
2. 상태: `제안됨` → `승인됨` → `대체됨` / `폐기됨`
3. 모든 팀원이 ADR을 검토하고, 필요 시 코멘트를 남깁니다
